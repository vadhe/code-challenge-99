# Problem 1 - Sum Algorithms

TypeScript implementation of three different approaches to calculate sum from 1 to n.

## How to Run

1. **Navigate to the problem-1 directory:**
   ```bash
   cd src/problem-1/
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the main script:**
   ```bash
   npm run dev
   ```

4. **Run tests:**
   ```bash
   npm run test
   ```
